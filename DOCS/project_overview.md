### Business Problem
The business needs a clear view of internet sales performance to track revenue trends, top customers, best-performing products, and compare actual sales against budget.

### Dataset Description
The project uses five tables:
- FACT_InternetSales
- DIM_Product
- DIM_Customer
- DIM_Calendar
- SalesBudget

### Approach
1. Performed data cleaning and validation in SQL Server
2. Designed a star schema with fact and dimension tables
3. Built KPIs and analytical queries
4. Created interactive dashboards in Power BI

### Dashboards Created
- Executive Sales Overview
- Customer Performance Analysis
- Product Performance Analysis

### Outcome
The dashboards help stakeholders monitor sales performance, identify growth opportunities, and make data-driven decisions.
